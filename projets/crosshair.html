Crosshair

Pratique pour les jeux de tirs, cette application dessine un petit réticule au centre de votre écran principal dans le but d’aider à la précision de ces jeux. Malgré que cette application soit écrite en C#, elle est offerte seulement sur les machines Windows, vu qu’elle utilise l’API de celui-ci pour créer une fenêtre et dessiner à l’intérieur. Le réticule peut s’adapter aux différents jeux : à l’aide d’une combinaison de touches, il est possible de changer l’endroit du dessin afin d’accommoder tous les jeux. Un fichier de configuration est aussi disponible pour automatiser le placement du réticule au lancement d’une application. Ce fichier utilise le format JSON.
